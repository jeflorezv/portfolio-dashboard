How to keep Stake & Bamboo values current
Neither Stake nor Bamboo offer a personal API — but here's the practical workflow
Weekly (2 min, Sundays)
1. Open Stake → check total value + any new shares
2. Open Bamboo → check portfolio value
3. Click "Weekly update" button → enter values → Save
Monthly (Stake CSV export)
Stake → Account → Statements → Download CSV. Use this to reconcile exact share counts after any purchases. Paste updated share counts into the Weekly Update tool.
